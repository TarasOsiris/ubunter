Installer

```shell
echo "Installing git..."
sudo apt-get update > /dev/null
sudo apt-get install -y git > /dev/null

echo "Cloning ubunter..."
git clone https://github.com/TarasOsiris/ubunter.git ~/.local/share/ubunter > /dev/null

source ~/.local/share/ubunter/install.sh
```
